# Devicon-rails
[![Gem Version](https://badge.fury.io/rb/devicon-rails.svg)](https://badge.fury.io/rb/devicon-rails)
## Installation

Add this line to your application's Gemfile:

```ruby
gem 'devicon-rails'
```

And then execute:

    $ bundle

In your application.css add this line:
```css
/*
 *= require devicon
 *= require devicon-colors
 */
```
